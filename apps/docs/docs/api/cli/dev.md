# dev

Starts a development server with live preview.

:::info Coming Soon
This command is currently in development.
:::

## Planned Usage

```bash
npx minopamotion dev <entry-point> [options]
```

## Options

| Flag | Description |
|------|-------------|
| `--port` | Dev server port (default: 3000) |

## Example

```bash
npx minopamotion dev src/index.tsx --port 4000
```

Opens the Studio interface in the browser for previewing compositions.
